# 🌐 My Link Hub

A modern and responsive personal Linktree-style website built with **Next.js**, **TypeScript**, and **Tailwind CSS**. It serves as a central hub to showcase my portfolio, social profiles, and contact information with a clean, elegant, and animated user interface.

## ✨ Features

- 🎨 Modern glassmorphism UI
- 📱 Fully responsive design
- ⚡ Built with Next.js App Router
- 🌙 Beautiful gradient background
- 👤 Personal profile section
- 🔗 Social media links
- 💼 Portfolio showcase
- 📧 Contact section
- ✨ Smooth hover animations
- 🚀 Fast performance and SEO-friendly

## 🛠️ Tech Stack

- Next.js 16
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Lucide React Icons

## 🚀 Getting Started

### Clone the repository

```bash
git clone <repository-url>
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

Open **http://localhost:3000** in your browser.
